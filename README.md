# クイズ採点機

クイズ研究会(サークル)で使用するクイズ採点機(現在は多答形式のみを実装予定。今後一問一答も追加)
回答者の順位と点数が表示されます

## 使い方

1. 正答、回答者の名前がつけられたtxtファイルを用意する
2. index.htmlを開く
3. 左に正答、右に回答者のファイルをすべて入力する
4. 採点ボタンを押す

### ファイルの書き方

答えを一行にひとつ書く

余計な空白や改行を含めないようにしてください(今後対応予定)

例

正答

```txt:正答.txt
しゅはり
ヒキュウ
繁田
```

回答

```txt:回答者1.txt
しゅはり
ヒキュウ
にぼ次郎
```

## 動作確認環境

Windows10

Chrome
